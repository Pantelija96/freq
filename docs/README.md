# Freq Guides

Available guides:

- `frontend-user-guide.html`
- `backend-admin-guide.html`

How to use them:

1. Open the HTML file in a browser.
2. Use the browser print dialog.
3. Choose `Save as PDF`.

Both guides use the Freq logo and the Android app color palette:

- `#0b1731`
- `#12284f`
- `#1b335f`
- `#3e5ea7`
